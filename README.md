# CySA+ Practice Exam App (CS0-003)

A browser-based practice app designed to help prepare for the CompTIA CySA+ certification exam.

## 🚀 Features

* 1000+ practice questions
* Practice mode with instant feedback
* Exam simulation mode (85 questions)
* Flashcard mode
* Search and filter questions by domain
* Score tracking (stored locally in browser)

## 📊 Exam Domains Covered

* Security Operations (33%)
* Vulnerability Management (30%)
* Incident Response (20%)
* Reporting & Communication (17%)

## 🌐 Live App

👉 https://github.com/elienzweme/cysa-plus-practice-app/blob/main/index.html
  
## 🖥️ How to Run Locally

1. Download the repository
2. Open `index.html` in any browser

This project was built as a hands-on study tool and to demonstrate practical frontend and cybersecurity knowledge.
